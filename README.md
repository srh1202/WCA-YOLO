# WCA-YOLO
Wavelet Pooling-based CSP Attention YOLO

Dataset
HVAC temperature detection dataset can be obtained from the following link:
https://drive.google.com/file/d/1XUqFOKj5UCjFcdHzF1ejS6DR3ZRp0ui7/view?usp=sharing

**Usage Instructions:**
After downloading the HVAC temperature detection dataset, place it in the root directory of the code. Use the command **python train.py** to start training. Modify the dataset path specified in the detect.py file, and run **python detect.py** to obtain the detection results.
